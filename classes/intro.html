HTML Attributes

  • Sometimes we need to include additional information in our tags.
  • We can include this information by defining the values of attributes.

HTML Attributes For Links

  • The text content of a link goes inside of the <a> tag.
  • <a>click this link</a>
  • If you click this link you will notice it doesn't take you anywhere... We need to define an attribute in the tag for the destination.

The href attribute

  • Attributes are added to tags like this:
  • <a href = "http://google.com">click this link</a>
  • If you click this link you will notice it will take you to Google.
  • The name of the attribute goes inside the tag, followed by an equals sign, then the value in quotes.

The style attribute

  • HTML doesn't give us much control over the appearance of our document.
  • In order to style our HTML, we need to use CSS.
  • One way to do this is to add the style attribute to your HTML element

The style attribute

  • A basic header tag:
  • <h1>Hello World!</h1>
  • With a style attribute to make the text color red:
  • <h1 style = "color: red">Hello World!</h1>

Multiple Attributes

  • An HTML tag can have many attributes
  • The attributes can be in any order in the tag
  • <a href = "http://google.com" style = "color: red">click this link</a>
  • Or...
  • <a style = "color: red" href = "http://google.com">click this link</a>

CSS

Cascading Style Sheets

CSS Syntax

Selector, Property, Value.

Property : Value

  • Recall the style we applied to the text in the previous exercises:
  • <h1 style = "color: red">Hello World!</h1>
  • In this example, the property of the element we are modifying is its color and we are setting it to red
  • We can modify more than one property : value pair by separating them with semicolons:
  • <h1 style = "color: red; background-color: white">Hello World!</h1>
  • In this example we have modified the color and background-color so they are red and white
