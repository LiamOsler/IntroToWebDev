Introduction to coding for the web

Lesson Assignment

Instructions

1. Codepen

  • Create an account on codepen.io
  • Create a new empty pen. In the sidebar menu, under the "create" section, click the word "Pen". This will create a empty pen and forward you to the codepen editor.

2. Basic HTML

  • In the "HTML" section of the codepen editor, add the following:
    • A level one h1 heading tag that contains your name.
    • Below that, a Paragraph tag with the text My Favourite Hobbies Are:
    • Below that, an Unordered list with at least three list items.
    • Below that, a h2 tag with the text My Web Presence:

3. Making a link

  • Below your h2 tag:
    • Add a hyperlink using the a tag.
    • Set the text of this link to be: My Dal CS Website
    • Set the href attribute of this link to be: https://web.cs.dal.ca/~csid/ where csid is your Computer Science faculty ID.

4. Basic CSS

  • Add a style attribute to the heading with your name you created earlier. Set the color of the text in this element to red.

5. CSS Selectors

  • Create a <style> </style> tag in your HTML.
  • In this style tag, use a selector to make all li elements a different color than the browser default.

6. Classes and IDs

  • Add a class attribute called big-heading to the h1 and h2 tags.
  • Select this class inside your <style> </style> tag using .big-heading{ } and set the font-weight to 900
  • Add an id attribute called hobbies-list to the unordered list tag.
  • Select this id inside your <style> </style> tag using #hobbies-list{ } and set the font-weight to bold

7. Questions and Answers

  • In a new paragraph tag at the end of your pen, please write a short answer to the following question: What is the difference between an ID and a Class in CSS?

Submission

  • Create a .txt file where the file name is your student number. For instance, if my student number is B00123456, then my file name is B00123456.txt
  • Copy and paste the URL for your pen in to this .txt file. You can take the URL for the pen directly from the address bar of your browser when you are editing the pen. Your URL for your pen will look something like this: https://codepen.io/liamosler/pen/eYropQY
  • Submit your .txt file to the submission box for this assignment on the Online Learning Platform. Do not include anything else other than the Codepen URL in the .txt file. Please ensure that your filename matches your student ID correctly.

Marking

  • One point will be allocated for the successful completion of each numbered step in the assignment instructions. Accordingly, this assignment is marked out of 7.
  • Please ensure in advance you are able to successfully create a Codepen and understand how to create a new Codepen.
  • Make sure the pen's visibility is set to public. If it is private, it cannot be evaluated. Once you receive a mark for the assignment, you can set the pen's visibility to private if you wish. Do not delete evaluated Codepens until the end of the course.
